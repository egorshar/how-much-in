- React Intl
- Tailwind (twrnc)
- Zustand

## Env

- node v16.15.1
- yarn v1.22.15
- `sudo npm i -g eas-cli expo-cli`

## Dev Setup
```sh
# run once
yarn install
eas device:create
yarn dev:ios
yarn dev:android

# next time run only
yarn start
```

## Production build
```sh
yarn build:ios
yarn build:anroid
```
